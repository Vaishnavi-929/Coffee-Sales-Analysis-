# Coffee-Sales-Analysis-
# Description-
Coffee Sales Dashboard (Excel)
Developed an interactive Coffee Sales Dashboard in Microsoft Excel to analyze sales performance. The dashboard provides insights into coffee types, payment methods, hourly and monthly sales trends, weekdays, and time-of-day sales using Pivot Tables, Pivot Charts, Slicers, and KPIs. It helps identify customer purchasing patterns and supports data-driven business decisions.
